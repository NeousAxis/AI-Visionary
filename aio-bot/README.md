# AIO Bot - v1.0.0

AIO Bot is an Artificial Intelligence Optimization tool designed to prepare businesses for the age of AI Search (SGE, ChatGPT, Gemini).

## Features
- **AI Visibility Audit**: Scores readability, credibility, and authority for AI agents.
- **Action Plan**: Prioritized list of corrections.
- **Content Generation**: Automatically generates JSON-LD, FAQ datasets, and Glossaries.
- **Export**: Downloads a full "AIO-READY" ZIP package.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS (Variables, Glassmorphism design system)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animation**: Framer Motion
- **Zip Generation**: JSZip

## Installation
1. Navigate to the folder:
   ```bash
   cd aio-bot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Usage
1. Open the local URL (http://localhost:5173).
2. Enter a Company Name and URL.
3. Watch the bot analyze and generating the report.
4. Review the scores and download the "AIO Ready" ZIP file.

## Note
This version runs in "Simulation Mode" for demonstration purposes as it does not rely on backend crawlers. The output structure and logic are production-ready schemas.
